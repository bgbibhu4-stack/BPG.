# Personal Resume Website - Bibhuprasad Gumansingh

A modern, responsive personal portfolio website designed to showcase education, skills, and projects.

## Project Overview

This website features a clean, professional design with smooth animations and a user-friendly interface. It includes sections for:
*   **About Me**: Personal introduction and bio.
*   **Education**: Details about your Diploma in DPMT and schooling.
*   **Skills**: Technical and soft skills, including software proficiency.
*   **Contact**: Information for potential employers to reach out.

## Key Features

*   **Dark/Light Mode**: Toggle between a premium Dark Navy theme and a clean Light theme using the Sun/Moon icon in the header.
*   **Responsive Design**: Optimized for mobile, tablet, and desktop viewing.
*   **Animations**: Smooth fade-in effects on scroll.
*   **Profile Integration**: Displays your personal photo in the About section and header logo.

## File Structure

*   **`index.html`**: The main structure of the website. Double-click this file to view your site.
*   **`styles.css`**: Contains all the styling rules, including colors, fonts, and the dark/light mode themes.
*   **`script.js`**: Handles interactivity like the theme toggle, scroll animations, and header behavior.
*   **`profile.jpg`**: Your personal profile photo used on the website.
*   **`README.md`**: This documentation file.

## How to Use

1.  **View the Website**: simply double-click the `index.html` file in this folder. It will open in your default web browser.
2.  **Edit Content**: You can open `index.html` in a text editor (Notepad, VS Code) to update your text.
3.  **Update Photo**: Replace `profile.jpg` with a new image of the same name to update your picture.

## Customization

*   **Colors**: You can change the color themes by editing the CSS variables in `styles.css`.
*   **Text**: Update your information directly in `index.html`.
